# 图标源文件

这个文件夹存放 PetGPT 的**原始图标文件**。

## 必需文件

### 1. `app-icon.png`
- **用途**: 应用程序图标（Dock、Finder、启动器等）
- **尺寸**: 1024x1024 像素或更大
- **格式**: PNG，透明背景
- **设计**: 完整的品牌设计，可以包含复杂细节

### 2. `tray-icon.png`
- **用途**: 菜单栏/系统托盘图标
- **尺寸**: 128x128 像素或更大
- **格式**: PNG，透明背景
- **设计要求**:
  - ⚠️ **必须与应用图标不同**
  - ✅ 简洁设计（托盘图标很小，16-44像素）
  - ✅ 粗线条（2-3px）
  - ✅ 高对比度
  - ✅ 建议使用单色或简单配色

## 使用方法

### 第一次使用

1. 将你的原始图标文件放到这里：
   ```
   design/icons/
   ├── app-icon.png      # 你的应用图标
   └── tray-icon.png     # 你的托盘图标
   ```

2. 运行生成脚本：
   ```bash
   sh scripts/generate-all-icons.sh
   ```

3. 所有需要的图标会自动生成到 `src-tauri/icons/`

### 更新图标

替换此文件夹中的 PNG 文件，然后重新运行：
```bash
sh scripts/generate-all-icons.sh
```

## 设计建议

### 应用图标（app-icon.png）
```
🎨 完整、详细的设计
- 可以使用多种颜色
- 可以包含细节和阴影
- 适合在 128x128 或更大尺寸显示
```

### 托盘图标（tray-icon.png）
```
⚫ 简化、清晰的设计  
- 简单的形状
- 粗线条（容易识别）
- 最好是单色或双色
- 适合在 16x16 到 44x44 像素显示
```
